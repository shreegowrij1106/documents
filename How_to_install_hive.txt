1) First, download Hive from the following link:
http://www.apache.org/dyn/closer.cgi/hive/
2) Open command prompt, go to the folder where you have download Hive and run the following command to extraxt files.
$ tar -xvf apache-hive-3.1.1-bin.tar.gz
3) Add Hive path to the environment variable. Open the environment variable window by searching it in the start menu
HIVE_HOME = D:\softwares\Hive\apache-hive-3.1.2-bin

	https://archive.apache.org/dist/db/derby/db-derby-10.12.1.1/
6) set DERBY_HOME path
